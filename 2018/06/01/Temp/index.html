Something to Evernote

  • IDEA
    • 注意 IDEA 对 Spring 的 XML 配置文件操作的删除 xsd 约束文件现象！
    • 211171974302 - 530267
    • https://www.jetbrains.com/help/idea/sending-feedback.html
    • Alt + F7 快捷键：查找类，方法，字段在项目中的调用情况
    • Ctrl + F7 快捷键：查找类，方法，字段在当前文件中的调用情况
    • Ctrl + Shift + I 快捷键：在类，方法，字段处弹窗查看定义逻辑
    • Ctrl + Q 快捷键：在类，方法，字段处显示文档和头信息，头信息比如类定义，方法定义，字段定义
    • Ctrl + Alt + U 展示类的 UML 图，需要在单独的新标签显示加上 Shift.
    • IDEA 的 Git 操作：在本地创建分支，合并分支，解决冲突！！！
    • Shift + Alt + F10：打开 Run 弹窗，选择对应的 Run 或者 Debug 或者插件的相关运行选项。
    • Shift + Alt + F 将类或者文件加入收藏，后面用 Alt + 2 打开目录。
